1) Images

All the images you can see in the template can be found on http://flickr.com/ None of the photographs you can see in the template, can be downloaded along the template files. All images have appropriate licenses that allow us to show them in the template preview.

2) Fonts

This template features 1 font. The font used is "Lato" in various weights (Regular, semi-bold, bold) and it is available for free download in the Google fonts directory: http://google.com/webfonts/. It can be implemented using the @font-face css option.
